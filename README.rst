Install
------------------------------------------------------------------------------

``learn_snowflake`` is released on PyPI, so all you need is to:

.. code-block:: console

    $ pip install learn-snowflake

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade learn-snowflake
